# Personal Productivity Agent

## Overview
A productivity management system built using FastAPI, Streamlit, SQLite, and SQLAlchemy.

## Features
- TASK MANAGEMENT
- Create Tasks
- View Tasks
- Complete Tasks
- Morning Check-in
- Evening Check-in
- Overdue Tasks
- End Of Day Summary
- Tomorrow Planner
- Weekly Review
- Dashboard Analytics

## Tech Stack
- Pandas

Programming Language:
- Python

Frontend:
- Streamlit

Backend:
- FastAPI

Database:
- SQLite
- SQLAlchemy

## Run Backend
uvicorn backend.main:app --reload

## Run Frontend
streamlit run frontend/app.py
